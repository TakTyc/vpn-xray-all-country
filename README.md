# 🌍 Xray VPN Configs – Sorted by Country

A handpicked, organized collection of **Xray VPN** configuration files (VLESS + XTLS) grouped by server location.  
Perfect for quick access, fallback routing, or multi-region testing.

---

## 📂 Structure

All configs are stored in country-named folders:
📁 configs/
├── 🇺🇸 US/
├── 🇬🇧 GB/
├── 🇩🇪 DE/
├── 🇫🇷 FR/
├── 🇯🇵 JP/
├── 🇸🇬 SG/
├── 🇦🇪 AE/
└── ...

text

Each folder contains:
- `config.json` – ready-to-use Xray client config
- `link.txt` – VLESS share link (if applicable)

---

## 🚀 How to Use

1. Clone this repo:
   ```bash
   git clone https://[https://github.com/TakTyc/vpn-xray-all-country]
Choose your preferred country folder.

Import config.json into your Xray client (e.g., v2rayN, Nekoray, Shadowrocket).

Connect and enjoy!

⚠️ Configs are provided as-is. Some may expire over time – check updates regularly.

📅 Update Frequency
Weekly refreshes (manually verified).

New countries added upon request.
